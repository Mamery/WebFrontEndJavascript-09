Bish Bosh 2.0

How to use:

The user enters a low number (A) and a higher number (B), and another number as a higher limit, and the program will print Bish for all numbers divisible by A, Bash for all numbers divisible by B, and Bish-Bash for all numbers divisible by both A and B util the limit number is reached. If the number is not divisible with neither A, B , A*B, the program will print the number.The number has to be a positive number!
